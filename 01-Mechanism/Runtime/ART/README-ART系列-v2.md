# ART 深度解析系列 · 系列总览 v2

> **本系列定位**：面向资深 Android 稳定性架构师，把"ART（Android Runtime）"——这个**承载所有 App 字节码执行、GC、类加载、JNI、信号处理、启动流程**的运行时核心，拆成 **9 大子模块（含 GC 9 大子系列，共 ~30+ 篇规划）**的长文。
>
> **基线声明**（本规范硬要求 · 用户 2026-07-17 决策升级）：
> - **AOSP**：`android-17.0.0_r1`（API 37，2026 Q2/Q3 发布；`refs/heads/android17-release` 或 `android-latest-release` manifest）
> - **Linux 内核**：`android17-6.18`（基于 Linux 6.18 LTS，2025-11-30 发布，EOL 2030-07-01）
> - **存量旧文基线**：AOSP 14 + android14-5.10/5.15（保留不重写，标"v1 旧稿，待基线升级"）
>
> **目录位置**：`Runtime/ART/`
>
> **作者决策日志**（本规范 §7 强制）：
>
> | 轮次 | 类别 | 决策 | 理由 | 影响范围 |

## 1. ART 全景与稳定性价值

### 1.1 ART 在 Android 栈中的位置

```
┌──────────────────────────────────────────────────────┐
│  Android 应用（Java/Kotlin）                          │
│  ★ App 主线程 / 系统服务                              │
└────────────────────────┬─────────────────────────────┘
                         ▼
┌──────────────────────────────────────────────────────┐
│  Framework（API 32+）                                │
│  ★ ActivityManager / WindowManager / PowerManager   │
└────────────────────────┬─────────────────────────────┘
                         ▼
┌──────────────────────────────────────────────────────┐
│  ART 运行时（本系列主题 · 9 大子模块）                │
│  ★ 字节码 / 编译 / 类加载 / GC / JNI / 信号 / 启动   │
│  ★ Android 17 新增：无锁 MessageQueue、static final │
│    不可变、ART 分代 GC 强化、AppFunctions            │
└────────────────────────┬─────────────────────────────┘
                         ▼
┌──────────────────────────────────────────────────────┐
│  Bionic libc + Linux Kernel（android17-6.18）       │
│  ★ Native 库 / 系统调用 / 进程管理                    │
└──────────────────────────────────────────────────────┘
```

**图 1-1 关键解读**：ART 是 Android 应用与内核之间的"运行时中间层" —— **所有 App 字节码都跑在 ART 上**。

### 6.2 工程默认值表（ART 通用）

| 参数 | 典型默认 | 选用准则 | 踩坑提醒 |
|------|---------|---------|---------|
| **ART GC 类型** | 分代 CC（Android 10+）| Android 17 强化分代 | 旧 CMS GC 已 deprecated |
| **JIT 编译阈值** | 10000 次方法调用 | 视启动期 vs 稳态权衡 | 太低 → 启动期卡 |
| **AOT 编译（dex2oat）** | 安装时 | 视存储 vs 启动权衡 | 太大 → 安装慢 |
| **类加载并行** | 默认开启 | — | OEM 改 ClassLoader 容易踩 |
| **MessageQueue（Android 17）** | 无锁（API 37+）| Android 17 应用 | 反射私有字段会崩 |
| **static final（Android 17）** | 不可变（API 37+）| Android 17 应用 | 反射/JNI 改会崩 |
| **堆大小** | 256MB-512MB（设备）| 视设备 RAM | 太小→频繁 OOM；太大→GC 慢 |
