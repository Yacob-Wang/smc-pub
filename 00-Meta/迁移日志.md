# 迁移日志（00-Meta/迁移日志）

> **目标**：将 smc-pub 目录重构为 **6 大分类架构**（AOSP 系统分层 + 工作流双轴）

## 架构总览

```
smc-pub/
├── 00-Meta/                  元信息 + 构建产物（地图）
├── 01-Mechanism/             机制（按 AOSP 分层）
├── 02-Symptom/               症状（按 S 编号 11 篇）
├── 03-Forensics/              取证（按 F 编号对齐 S 编号）
├── 04-Tool/                  工具
├── 05-Governance/            治理（运营视角）
├── 06-Case/                  案例
└── 06-Foundation/            基础（BSP / 构建 / 杂项）
```

**6 大分类角色**：
- **00-Meta**：项目地图 / 引用矩阵 / 术语表 / 版本基线
- **01-Mechanism**：按 AOSP 系统分层理解原理
- **02-Symptom**：7 大症状机制（S01-ANR ~ S10-Measure）
- **03-Forensics**：7 大症状取证（F00-F07，对齐 S 编号）
- **04-Tool**：7 大工具（Dumpsys / Watchdog / Perfetto / Hprof / AmCommand / ANR-Detection / Tracing）
- **05-Governance**：8 大治理主题（APM / OEM / CrossPlatform / LowEnd / AI / PerfMem / Security）
- **06-Case**：启动场景案例（E01-E11）
- **06-Foundation**：基础（BSP / 构建 / 集成 / 杂项）
